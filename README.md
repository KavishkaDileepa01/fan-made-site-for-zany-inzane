# fan-made-site-for-zany-inzane

Unofficial fan-made tribute website for Sri Lankan rap artist Zany Inzane.

This project is **not affiliated with the artist or his management**. It links only to public official channels on YouTube, Facebook, and Instagram.

## Live site

https://kavishkadileepa01.github.io/fan-made-site-for-zany-inzane/

## If Chrome shows a security warning

Google Safe Browsing can temporarily flag fan sites that look similar to official artist pages.

1. Deploy the latest version of this site.
2. Report a false positive: https://safebrowsing.google.com/safebrowsing/report_error/
3. Verify the site in [Google Search Console](https://search.google.com/search-console) and request a review under **Security issues**.
